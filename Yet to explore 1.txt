Kafka
Kafka is a distributed event‑streaming platform that stores and processes high‑volume, real‑time data using partitioned, replicated logs.
It enables scalable, fault‑tolerant event pipelines and allows consumers to replay events at any time.

MSMQ
MSMQ is a Windows‑based message‑queueing system that provides reliable, asynchronous communication between applications using disk‑backed queues.
It ensures guaranteed delivery even when the sender or receiver is offline, but is considered legacy today.

Octopus Deploy
Octopus Deploy is a deployment automation and release‑management platform that orchestrates multi‑environment deployments using versioned releases and variable management.
It integrates with CI tools to deploy artifacts consistently across Dev, QA, UAT, and Prod.

Redis
Redis is an in‑memory key‑value data store used for ultra‑fast caching, session storage, real‑time operations, and lightweight messaging.
It achieves microsecond latency by keeping all data in RAM and supporting rich data structures like lists, sets, and streams.
